# Protocol State Machines

## v1 (AGIJobManager)

### States
`DISCOVERED`, `EVALUATED`, `APPLY_READY`, `APPLIED_UNSIGNED`, `APPLIED_CONFIRMED`, `EXECUTION_READY`, `EXECUTED`, `VALIDATED`, `PUBLISHED`, `SUBMISSION_READY`, `SUBMITTED_UNSIGNED`, `RECONCILING`, `DONE`, `FAILED`, `BLOCKED_RETRIEVAL_REQUIRED`, `BLOCKED_OPERATOR_REVIEW`, `BLOCKED_ARTIFACT_EMISSION_REQUIRED`.

### Legal transitions
- `DISCOVERED -> EVALUATED | BLOCKED_RETRIEVAL_REQUIRED | FAILED`
- `EVALUATED -> APPLY_READY | FAILED`
- `APPLY_READY -> APPLIED_UNSIGNED | BLOCKED_OPERATOR_REVIEW | FAILED`
- `APPLIED_UNSIGNED -> APPLIED_CONFIRMED | BLOCKED_OPERATOR_REVIEW | FAILED`
- `APPLIED_CONFIRMED -> EXECUTION_READY | FAILED`
- `EXECUTION_READY -> EXECUTED | BLOCKED_RETRIEVAL_REQUIRED | FAILED`
- `EXECUTED -> VALIDATED | FAILED`
- `VALIDATED -> PUBLISHED | FAILED`
- `PUBLISHED -> SUBMISSION_READY | BLOCKED_ARTIFACT_EMISSION_REQUIRED | FAILED`
- `SUBMISSION_READY -> SUBMITTED_UNSIGNED | BLOCKED_OPERATOR_REVIEW | FAILED`
- `SUBMITTED_UNSIGNED -> RECONCILING | BLOCKED_OPERATOR_REVIEW | FAILED`
- `RECONCILING -> DONE | FAILED`

### Blocked states and causes
- `BLOCKED_RETRIEVAL_REQUIRED`: retrieval packet missing/invalid before solve stage.
- `BLOCKED_OPERATOR_REVIEW`: unsigned tx exists but explicit operator action required.
- `BLOCKED_ARTIFACT_EMISSION_REQUIRED`: canonical artifact bundle incomplete before submission/review boundaries.

## Prime (AGIJobDiscoveryPrime)

### States
`DISCOVERED`, `EVALUATED`, `COMMIT_READY`, `COMMITTED_UNSIGNED`, `COMMIT_CONFIRMED`, `REVEAL_READY`, `REVEALED_UNSIGNED`, `REVEAL_CONFIRMED`, `SHORTLIST_PENDING`, `FINALIST`, `FINALIST_ACCEPT_READY`, `FINALIST_ACCEPTED_UNSIGNED`, `FINALIST_ACCEPT_CONFIRMED`, `TRIAL_EXECUTION_READY`, `TRIAL_EXECUTED`, `TRIAL_PUBLISHED`, `VALIDATOR_SCORING_PENDING`, `WINNER_PENDING`, `WINNER_DESIGNATED`, `FALLBACK_PROMOTABLE`, `DONE`, `FAILED`, `BLOCKED_DEADLINE`, `BLOCKED_OPERATOR_REVIEW`, `BLOCKED_RETRIEVAL_REQUIRED`, `BLOCKED_ARTIFACT_EMISSION_REQUIRED`.

### Legal transitions
- `DISCOVERED -> EVALUATED | BLOCKED_RETRIEVAL_REQUIRED | FAILED`
- `EVALUATED -> COMMIT_READY | BLOCKED_DEADLINE | FAILED`
- `COMMIT_READY -> COMMITTED_UNSIGNED | BLOCKED_OPERATOR_REVIEW | BLOCKED_DEADLINE | FAILED`
- `COMMITTED_UNSIGNED -> COMMIT_CONFIRMED | BLOCKED_OPERATOR_REVIEW | FAILED`
- `COMMIT_CONFIRMED -> REVEAL_READY | BLOCKED_DEADLINE | FAILED`
- `REVEAL_READY -> REVEALED_UNSIGNED | BLOCKED_OPERATOR_REVIEW | BLOCKED_DEADLINE | FAILED`
- `REVEALED_UNSIGNED -> REVEAL_CONFIRMED | BLOCKED_OPERATOR_REVIEW | FAILED`
- `REVEAL_CONFIRMED -> SHORTLIST_PENDING | FAILED`
- `SHORTLIST_PENDING -> FINALIST | WINNER_PENDING | FAILED`
- `FINALIST -> FINALIST_ACCEPT_READY | BLOCKED_DEADLINE | FAILED`
- `FINALIST_ACCEPT_READY -> FINALIST_ACCEPTED_UNSIGNED | BLOCKED_OPERATOR_REVIEW | BLOCKED_DEADLINE | FAILED`
- `FINALIST_ACCEPTED_UNSIGNED -> FINALIST_ACCEPT_CONFIRMED | FAILED`
- `FINALIST_ACCEPT_CONFIRMED -> TRIAL_EXECUTION_READY | FAILED`
- `TRIAL_EXECUTION_READY -> TRIAL_EXECUTED | BLOCKED_DEADLINE | FAILED`
- `TRIAL_EXECUTED -> TRIAL_PUBLISHED | FAILED`
- `TRIAL_PUBLISHED -> VALIDATOR_SCORING_PENDING | FAILED`
- `VALIDATOR_SCORING_PENDING -> WINNER_PENDING | BLOCKED_DEADLINE | FAILED`
- `WINNER_PENDING -> WINNER_DESIGNATED | FALLBACK_PROMOTABLE | FAILED`
- `FALLBACK_PROMOTABLE -> WINNER_DESIGNATED | BLOCKED_DEADLINE | FAILED`
- `WINNER_DESIGNATED -> DONE | FAILED`

### Deadline guard model
- Commit window: `commitStart <= now < commitDeadline`
- Reveal window: `commitDeadline <= now < revealDeadline`
- Finalist acceptance window: `revealDeadline <= now < finalistAcceptDeadline`
- Trial window: `finalistAcceptDeadline <= now < trialDeadline`
- Score commit window: `trialDeadline <= now < scoreCommitDeadline`
- Score reveal window: `scoreCommitDeadline <= now < scoreRevealDeadline`
- Fallback promotion window: `now >= scoreRevealDeadline`

### Blocked states and causes
- `BLOCKED_DEADLINE`: current action attempted outside legal phase window.
- `BLOCKED_OPERATOR_REVIEW`: human approval/signing boundary reached.
- `BLOCKED_RETRIEVAL_REQUIRED`: retrieval packet missing for protocol-correctness stage.
- `BLOCKED_ARTIFACT_EMISSION_REQUIRED`: canonical artifact bundle incomplete.
