# Operator Instructions (Safety Entry Point)

Use this as the shortest safe-operating guide.

## Absolute rules
1. Never sign a transaction you cannot decode and verify.
2. Never rely on runtime auto-send behavior (there is none by doctrine).
3. Only sign after artifact bundle + checklist review.
4. If artifacts/state disagree, stop and resolve before action.

## What you review before any signature
- Correct contract address and chain ID.
- Decoded function call + arguments.
- Artifact bindings (the files this tx commits to).
- Preconditions/checklist completion.
